# プロダクト概要

## 作るもの

アプリ名は `タイ文字ノート`。

日本語話者向けに、タイ文字の基礎からタイ語をしっかり学べるWebアプリ。

初心者が、タイ文字・独自ローマ字表記・声調・基本単語を段階的に学べることを目指す。

## 想定ユーザー

- タイ語を初めて学ぶ日本語話者
- タイ文字を読めるようになりたい人
- 独学で発音や声調につまずいている人
- 会話フレーズを丸暗記する前に、文字と音の基礎を固めたい人
- タイ旅行やタイ移住に向けて、最低限の読み方と会話力を身につけたい人

## 価値

タイ語は、文字・発音・声調が日本語話者にとって難しい。

このアプリでは、単語暗記だけでなく、以下を丁寧に扱う。

- タイ文字の形と音
- 子音・母音の組み合わせ
- 子音クラスと声調の基礎
- 声調の違い
- 独自ローマ字表記を使った読み方
- 日本語との違い

## 学習の軸

最優先は、タイ文字を覚えるための基礎学習。

会話や単語も扱うが、最初は「文字を見て独自ローマ字表記と音の特徴がわかる」「中子音・高子音・低子音の分類がわかる」状態を作ることを重視する。

読み方は基本的にカタカナを使わず、独自ローマ字表記を中心に扱う。

IPAは必要に応じて補助的に併記できるようにするが、MVPでは必須にしない。

最初のMVPでは音声なしで作り、文字・独自ローマ字表記・分類・クイズの学習体験を先に固める。

## 最初のMVP

最初のバージョンでは、機能を増やしすぎず、次の学習体験に集中する。

1. ホーム画面で今日の学習を始める
2. タイ文字カードで形と音を確認する
3. 独自ローマ字表記と子音分類を確認する
4. 文字・独自ローマ字表記・分類・意味を4択クイズで確認する
5. 正解・不正解と進捗が保存される

## 対応範囲

### 最初に入れる

- タイ文字の基礎
- 子音・母音・声調の入門
- 中子音・高子音・低子音の分類
- 基本文字のフラッシュカード
- 文字と独自ローマ字表記の対応クイズ
- 子音分類クイズ
- 基本単語
- フラッシュカード
- 4択クイズ
- 学習進捗

### 後から検討する

- 発音録音と判定
- 音声再生
- 手書き練習
- AI会話練習
- PWA化
- スマホアプリ化
- ユーザーアカウント
- 有料プラン

## トーン

落ち着いた学習ツールとして、学習者を焦らせず、短時間で少しずつ進められる雰囲気にする。

「完璧に覚える」よりも、「毎日少しずつ文字と音に慣れる」体験を重視する。
