# 学習コンテンツ設計

## 基本方針

日本語話者がつまずきやすい順に、短い単位で学べるようにする。

最初はタイ文字を覚えるための土台を重視する。

「文字」「独自ローマ字表記」「子音分類」「声調」「単語」を結びつけ、文字を見たときに読み方と音の特徴を思い出せる状態を目指す。

読み方は基本的にカタカナを使わず、学習者向けの独自ローマ字表記を中心にする。

IPAは必要に応じて併記できるようにするが、MVPでは必須にしない。

ただし、`ึ` / `ื` は `ue` / `uue` ではわかりにくいため、表示では `ɯ` / `ɯː` を使う。

最初のMVPでは音声なしで作り、文字・独自ローマ字表記・分類・クイズの流れを先に整える。

## MVPの教材案

### レッスン1: タイ文字のしくみを知る

- タイ語には子音字、母音記号、声調記号がある
- 文字と独自ローマ字表記をセットで覚える
- タイ文字を見て種類を選ぶ4択クイズ

### レッスン2: 中子音から始める

- ก
- จ
- ด
- ต
- บ
- ป
- อ

### レッスン3: 高子音を増やす

- ข
- ฉ
- ถ
- ผ
- ส

### レッスン4: 低子音を増やす

- ค
- ง
- ช
- ท
- น

### レッスン5: 母音の基礎

- 短母音と長母音
- 文字の前後上下につく母音記号
- 独自ローマ字表記を見て母音を選ぶクイズ

### レッスン6: 声調に慣れる

- タイ語には声調がある
- 同じ音でも声調で意味が変わる
- 声調記号と子音分類の関係を学ぶ

### レッスン7: 文字から単語へ

- あいさつの単語
- 数字
- よく使う短い単語
- 文字、独自ローマ字表記、意味を結びつける

## 注意点

カタカナ読みは基本的に使わない。

タイ語の発音や声調は正確さが重要なため、独自ローマ字表記、子音分類、視覚的な説明を重視する。

音声はMVPでは入れないが、後から追加できるようにデータ構造には `audioPath` を残す。
