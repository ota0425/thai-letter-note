# 004: 発音表記は独自ローマ字表記を基本にする

## 状態

採用

## 決定

読み方は、カタカナではなく独自ローマ字表記を基本にする。

IPAは必須にしない。

ただし、将来的に必要になった場合は補助情報として併記できるようにする。

## 理由

カタカナ表記は日本語話者にとって見慣れているが、タイ語の母音、子音、声調を正確に表しにくい。

IPAは正確だが、初心者には記号の学習コストが高い。

そのため、最初は学習者向けの独自ローマ字表記を使い、必要に応じてIPAを追加できる形にする。

ただし、`ึ` / `ื` のように `ue` / `uue` では直感的にわかりにくい母音は、表示上はIPAに合わせて `ɯ` / `ɯː` を使う。

## データ方針

教材データでは、独自ローマ字表記とIPAを分けて持つ。

- `romanization`: 独自ローマ字表記。MVPで使う主表記。
- `ipa`: IPA表記。任意。

UIでは、まず `romanization` を表示する。

`ipa` は、詳細表示や上級者向け設定で使う可能性を残す。
