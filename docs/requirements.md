# 機能要件

## MVPで必要な機能

### ホーム

- 今日の学習を開始できる
- 学習中のレッスンがわかる
- 進捗が簡単に見える
- スマホ画面でも片手で主要操作ができる

### レッスン

- レベル別またはカテゴリ別にレッスンを表示する
- 各レッスンには説明、単語、例文、クイズを含める
- タイ語、日本語訳、独自ローマ字表記、分類を表示できる
- タイ文字の形、音、分類を確認できる

### タイ文字カード

- 子音字・母音記号・声調記号を表示する
- 文字の名前、独自ローマ字表記、分類を確認できる
- 子音字は中子音・高子音・低子音に分類して確認できる
- 書き順や形の特徴を説明できる余地を残す
- 似ている文字を比較できる余地を残す

### フラッシュカード

- タイ語の単語または文字を表示する
- 日本語訳、独自ローマ字表記、発音ヒントを確認できる
- 覚えた・まだ不安を記録できる

### クイズ

- 4択問題を出題する
- タイ文字から独自ローマ字表記を選ぶ問題を出題する
- 独自ローマ字表記からタイ文字を選ぶ問題を出題する
- タイ文字から子音分類を選ぶ問題を出題する
- タイ文字から日本語訳を選ぶ問題を出題する
- 正解・不正解を表示する
- 間違えた問題を復習対象にする
- 回答結果を進捗に反映する

### 進捗

- 完了したレッスンを保存する
- 正答率を保存する
- 復習が必要な項目を保存する

## タイ語学習として重要な機能

- 声調の違いを視覚的に示す
- 子音・母音・声調の関係を段階的に説明する
- 読み方は基本的にカタカナを使わず、独自ローマ字表記を使う
- IPAは補助情報として併記できる余地を残す
- 中子音・高子音・低子音を分類して覚えられるようにする
- 日本語話者が間違えやすい音を説明する

## 後から追加したい機能

- 音声再生
- 発音録音
- 発音の簡易フィードバック
- リスニング専用モード
- 書き取り練習
- タイ文字の手書き練習
- 会話シナリオ練習
- 学習リマインダー
- アカウント登録
- 複数端末での進捗同期

## 非機能要件

- スマホ画面で使いやすいこと
- PC、タブレット、スマホに対応するレスポンシブUIにすること
- モバイルファーストで主要画面を設計すること
- 将来的なPWA化を妨げない構成にすること
- 1回の学習が3分から5分で終わること
- オフラインでも一部学習できる余地を残すこと
- 学習データを増やしやすい構造にすること
- 将来的に音声ファイルを追加しやすい構造にすること
