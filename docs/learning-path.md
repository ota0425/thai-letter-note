# 学習カリキュラム

## 方針

`タイ文字ノート` では、基礎から応用まで体系的に学べる構成にする。

最初は文字の形と分類を覚え、次に独自ローマ字表記、母音、声調、単語、文、会話へ進む。

読み方は基本的にカタカナを使わず、学習者向けの独自ローマ字表記で扱う。

IPAは必要になった場合に補助情報として併記できるようにするが、MVPでは必須にしない。

## 全体ステップ

### Step 1: タイ文字の全体像

- 子音字、母音記号、声調記号の役割
- 文字が前後上下に組み合わさること
- タイ語には子音分類と声調があること

### Step 2: 子音分類

子音は一度に全部覚えず、分類ごとに少しずつ増やす。

- 中子音
- 高子音
- 低子音

各文字では、次を確認する。

- 文字
- 文字名
- 独自ローマ字表記
- IPA表記
- 子音分類
- 代表単語
- 似ている文字

### Step 3: 母音

- 短母音
- 長母音
- 特殊母音
- 前後上下につく母音記号
- 子音字との組み合わせ

### Step 4: 声調

- 声調の種類
- 声調記号
- 子音分類と声調の関係
- 生きた音節、死んだ音節

### Step 5: 基本単語

- 子音と母音の組み合わせから読める単語
- あいさつ
- 数字
- 日常でよく使う名詞・動詞・形容詞

### Step 6: 短い文

- 基本語順
- 否定
- 疑問
- 丁寧表現
- 短い例文の読解

### Step 7: 会話と応用

- あいさつ
- 自己紹介
- 買い物
- レストラン
- 移動
- 旅行で使う表現

## MVPの学習範囲

最初のMVPでは、Step 1からStep 3の入口までに絞る。

- タイ文字の全体像
- 中子音の一部
- 高子音の一部
- 低子音の一部
- 独自ローマ字表記との対応
- 子音分類クイズ
- 文字カード
- 進捗保存

## 子音の増やし方

一度に44文字を出さず、分類ごとに5文字前後から始める。

最初の候補:

- 中子音: ก, จ, ด, ต, บ, ป, อ
- 高子音: ข, ฉ, ถ, ผ, ส
- 低子音: ค, ง, ช, ท, น

学習者が分類と独自ローマ字表記に慣れてから、残りの子音を追加する。

## 発音表記

読みは独自ローマ字表記を中心にする。

カタカナ読みは、原則としてUIには出さない。

IPAは任意の補助情報として扱う。

教材データには、独自ローマ字表記を `romanization`、IPAを `ipa` として分けて持てるようにする。

## 音声

MVPでは音声なしで作る。

音声再生、リスニング、発音録音は後続フェーズで検討する。
