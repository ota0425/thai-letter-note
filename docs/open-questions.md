# 未決事項・確認したいこと

今後、決めたいことをここにまとめる。

## 決定済み

- アプリ名は `タイ文字ノート` を仮採用する
- 最優先はタイ文字を覚える基礎学習にする
- Webアプリから始める
- 最初からモバイル対応を意識したレスポンシブUIにする
- 雰囲気は落ち着いた学習ツールにする
- 基礎から応用まで体系的に学べる構成にする
- 読みは基本的にカタカナを使わず、独自ローマ字表記を使う
- IPAは任意の補助表記として扱う
- MVPでは音声なしで作る
- 子音は中子音・高子音・低子音に分類して、徐々に増やす

## プロダクト

- 主な対象は完全初心者か、少し学んだ人も含めるか
- 日本語の説明はどのくらい丁寧に入れるか

## 学習内容

- 最初に扱う中子音・高子音・低子音をそれぞれ何個にするか
- 母音や声調をどのタイミングで入れるか
- 独自ローマ字表記の細かいルールをどう定義するか
- 応用編でどこまで会話や文法を扱うか

## 機能

- ログインなしで始められる形にするか
- 進捗保存は端末内だけでよいか
- 音声再生をどのフェーズで追加するか
- 発音録音をどのフェーズで追加するか

## デザイン

- タイらしい色やモチーフをどの程度使うか
- 大人向けの落ち着いた雰囲気をどの程度強めるか

## 技術

- React + TypeScript + Viteで進めてよいか
- データは最初ローカル管理でよいか
- 将来的にバックエンドを使う予定があるか
